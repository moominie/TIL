## 리스트

### 순서 있는 리스트(1.) (Tab, shift + tab)

1. 식용유를 두른다.
2. 파를 넣고 파기름을 낸다.
3. 계란을 풀어서 익힌다.
4. 간장을 볶는다.
   1. 기름에 눌린다는 느낌으로...



### 순서 없는 리스트(-) (Tab, shift + tab)

- CLI 학습
  - 폴더
    - `mkdir`
    - `rm -f`
    - `cd`
  - 파일
    - `touch`
    - `rm`
    - `open`



## 인라인 코드(`)

파일을 생성하기 위해서 `touch` 명령어를 사용한다.

`cd`를 통해 폴더간 이동을 한다.



## 코드 블럭(```)

```
$ touch a.txt

$ rm a.txt

$ mkdir test

$ cd test
```
